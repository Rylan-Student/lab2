isOdd lab submission
--------------------
Files included:
- main.c     : Program entry; accepts one integer argument and reports whether it's odd.
- isOdd.c    : Implementation of isOdd().
- isOdd.h    : Header for isOdd().
- Makefile   : Builds the executable 'check_odd' and supports 'make clean'.

Build:
  make

Run:
  ./check_odd 42

Clean:
  make clean
